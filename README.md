## Project Structure

- **homepage.html**  
  Contains the main HTML structure of the website and also the setting for the homepage .

- **explore.html**  
  Contains the HTML structure of the explore page where users can view different festivals as cards.

- **about.html**  
  Contains the HTML structure of the about page which focuses on the mission of the website 

- **style.css**  
  Includes all the CSS styling used to design and format the home page.

- **style1.css**  
  Includes all the CSS styling used to design and format the explore page.

- **style2.css**  
  Includes all the CSS styling used to design and format the about page.

- **assets/**  
  A folder containing all image files and other media resources used in the project.

note that all the css file have consistent styling for repeting elements like the body, header and navigation styling.
